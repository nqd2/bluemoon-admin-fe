### PHẦN 1: TÀI LIỆU KỸ THUẬT (SRS) CHO TICKET [BE-05]

#### **Ticket: [BE-05] Get Residents List (Xem danh sách nhân khẩu)**

**1. Thông tin chung**

- **Assignee:** Backend Dev
- **Branch Name:** `feature/BE-05-get-residents`
- **Estimated Time:** 2h

**2. Mô tả (Description)**
API lấy danh sách cư dân, hỗ trợ các tính năng phân trang (pagination) và tìm kiếm theo tên (search).

**3. Yêu cầu kỹ thuật (Technical Specs)**

- **Endpoint:** `GET /api/residents`
- **Query Params:**
  - `page`: Số trang (default: 1)
  - `limit`: Số lượng item/trang (default: 10)
  - `keyword`: Từ khóa tìm kiếm theo tên (optional)
- **Logic:**
  - Sử dụng `regex` của MongoDB để tìm kiếm gần đúng (case-insensitive) cho `fullName`.
  - Sử dụng `.skip()` và `.limit()` để phân trang.

**4. Acceptance Criteria (Tiêu chí chấp nhận)**

- [x] **Data Structure:** Trả về object chứa:
  - `residents`: Array danh sách cư dân.
  - `page`: Trang hiện tại.
  - `pages`: Tổng số trang.
  - `total`: Tổng số bản ghi tìm thấy.
- [x] **Search:** Tìm kiếm hoạt động đúng với `keyword`.
- [x] **Pagination:** Phân trang hoạt động đúng.
- [x] **Populate:** Nếu cư dân đã thuộc căn hộ nào, populate thông tin cơ bản của `apartmentId` (ví dụ: `name`).
